## 功能特性

### 核心功能

| 功能 | 描述 |
|-----|------|
| 多市场支持 | 支持 A股、港股、美股三种市场 |
| 持仓管理 | 添加、编辑、删除股票持仓信息 |
| 实时股价 | 自动获取实时股价（A股、港股） |
| 数据可视化 | 多种图表展示持仓分析 |
| 数据导入导出 | 支持 JSON 和 CSV 格式 |
| 主题切换 | 支持明亮/暗黑主题 |
| 本地存储 | 所有数据保存在浏览器本地 |

### 图表类型

| 图表 | 用途 |
|-----|------|
| 饼图 | 展示各股票市值占比，了解持仓分布 |
| 柱状图 | 对比各股票盈亏率和盈亏金额 |
| 折线图 | 查看总市值的历史变化趋势 |
| 树形图 | 按行业分类查看持仓分布 |

## 快速开始

### 环境要求

- Node.js 18.0 或更高版本
- npm 或 pnpm 包管理器

### 安装步骤


## 使用指南

### 添加股票持仓

1. **选择市场**
   
   在左侧表单中，点击市场下拉框选择：
   - A股 - 中国大陆股票市场
   - 港股 - 香港股票市场
   - 美股 - 美国股票市场

2. **输入股票代码**
   
   根据选择的市场，输入对应格式的股票代码：
   
   | 市场 | 格式 | 示例 |
   |-----|------|------|
   | A股 | 6位数字 | 600519（贵州茅台）|
   | 港股 | 4-5位数字 | 00700（腾讯控股）|
   | 美股 | 1-5位字母 | AAPL（苹果）|

3. **填写持仓信息**
   
   | 字段 | 说明 | 是否必填 |
   |-----|------|---------|
   | 股票名称 | 可选，自动获取或手动输入 | 否 |
   | 买入价格 | 购买时的单价 | 是 |
   | 持仓数量 | 持有的股票数量（整数）| 是 |
   | 买入日期 | 购买日期 | 是 |
   | 当前价格 | 最新价格，可自动获取 | 否 |
   | 所属行业 | 用于行业分布分析 | 否 |

4. **提交保存**
   
   点击「添加持仓」按钮完成添加。

### 自动获取股价

#### 支持的市场

| 市场 | 自动获取 | 说明 |
|-----|---------|------|
| A股 | ✅ 支持 | 通过东方财富API获取 |
| 港股 | ✅ 支持 | 通过东方财富API获取 |
| 美股 | ❌ 不支持 | 请手动输入当前价格 |

#### 使用方法

1. 选择市场并输入股票代码
2. 点击「当前价格」输入框旁边的「自动获取」按钮
3. 系统会自动获取最新价格和股票名称
4. 如果获取失败，请检查股票代码是否正确，或手动输入价格

### 查看分析图表

添加持仓后，右侧会自动显示分析图表。

#### 饼图 - 持仓占比

- **用途**：了解各股票在总市值中的占比
- **解读**：
  - 扇区大小表示该股票市值占总市值的比例
  - 鼠标悬停可查看具体数值
  - 图例可点击隐藏/显示特定股票

#### 柱状图 - 盈亏对比

- **用途**：对比各股票的盈亏情况
- **解读**：
  - 绿色柱子表示盈利
  - 红色柱子表示亏损
  - 柱子高度表示盈亏率百分比

#### 折线图 - 市值趋势

- **用途**：查看总市值的历史变化
- **解读**：
  - 每次添加/修改持仓时自动记录当天市值
  - 最多保留30天的历史数据
  - 可观察投资组合的整体走势

#### 树形图 - 行业分布

- **用途**：按行业分类查看持仓分布
- **解读**：
  - 每个方块代表一个行业
  - 方块大小表示该行业的总市值
  - 需要在添加股票时填写「所属行业」

### 管理持仓

在持仓列表中，您可以：

| 操作 | 说明 |
|-----|------|
| 编辑 | 点击编辑按钮修改持仓信息，表单会自动填充当前数据 |
| 删除 | 点击删除按钮移除持仓，需确认后才会删除 |
| 刷新价格 | 点击刷新按钮获取该股票的最新价格 |

#### 编辑持仓

1. 找到要编辑的股票行
2. 点击「编辑」图标（铅笔图标）
3. 在左侧表单中修改信息
4. 点击「保存修改」

#### 删除持仓

1. 找到要删除的股票行
2. 点击「删除」图标（垃圾桶图标）
3. 在确认对话框中点击「删除」

### 数据导入导出

#### 导出数据

1. 在左侧面板找到「导出数据」按钮
2. 选择导出格式：
   - **JSON** - 适合备份和程序处理
   - **CSV** - 适合用 Excel 打开查看
3. 点击「导出数据」
4. 文件会自动下载，文件名包含当前日期

#### 导入数据

1. 点击「导入数据」按钮
2. 选择之前导出的 JSON 或 CSV 文件
3. 系统会验证数据格式并导入
4. 导入成功后会显示导入的数量

> ⚠️ **注意**：导入会覆盖当前所有持仓数据，请先备份。

### 主题切换

点击右上角的图标切换主题：

| 图标 | 主题 |
|-----|------|
| ☀️ 太阳 | 切换到明亮主题 |
| 🌙 月亮 | 切换到暗黑主题 |

主题选择会自动保存，下次打开时保持您的选择。

## 股票代码格式

### A股

| 类型 | 代码格式 | 市场代码 | 示例 |
|-----|---------|---------|------|
| 上海主板 | 6开头 | sh | 600519（贵州茅台）|
| 上海科创板 | 688开头 | sh | 688981（中芯国际）|
| 深圳主板 | 000开头 | sz | 000001（平安银行）|
| 深圳创业板 | 300开头 | sz | 300750（宁德时代）|

### 港股

| 类型 | 代码格式 | 示例 |
|-----|---------|------|
| 主板 | 4-5位数字 | 00700（腾讯控股）|
| 创业板 | 4-5位数字 | 08083（中国有赞）|

> 💡 **提示**：港股代码前面的0可以省略，如 00700 可以输入 700

### 美股

| 类型 | 代码格式 | 示例 |
|-----|---------|------|
| 纽约证券交易所 | 1-5位字母 | AAPL（苹果）、BRK.A（伯克希尔A）|
| 纳斯达克 | 1-5位字母 | MSFT（微软）、GOOGL（谷歌）|

> ⚠️ **注意**：美股暂不支持自动获取股价，请手动输入当前价格。

## 常见问题

### Q1: 为什么获取股价失败？

**可能原因：**

1. **股票代码错误** - 请检查代码格式是否正确
2. **网络问题** - 请检查网络连接
3. **API暂时不可用** - 请稍后重试
4. **美股不支持** - 美股需要手动输入价格

**解决方法：**

- 确认股票代码格式正确
- 手动输入当前价格

### Q2: 数据会丢失吗？

**数据存储位置：** 浏览器的 localStorage

**数据保留条件：**
- ✅ 刷新页面 - 数据保留
- ✅ 关闭浏览器后重新打开 - 数据保留
- ❌ 清除浏览器数据 - 数据丢失
- ❌ 使用无痕模式 - 关闭后数据丢失

**建议：** 定期使用导出功能备份数据

### Q3: 支持哪些浏览器？

| 浏览器 | 支持情况 |
|-------|---------|
| Chrome | ✅ 推荐 |
| Firefox | ✅ 支持 |
| Safari | ✅ 支持 |
| Edge | ✅ 支持 |
| IE | ❌ 不支持 |

### Q4: 港股和美股的盈亏如何计算？

系统会自动将不同货币转换为人民币（CNY）进行统一计算：

| 市场 | 货币 | 汇率（参考）|
|-----|------|-----------|
| A股 | CNY | 1 |
| 港股 | HKD | 0.92 |
| 美股 | USD | 7.24 |

> 汇率为固定参考值，可能与实际汇率有差异

### Q5: 如何在多台设备间同步数据？

目前不支持自动同步，但可以：

1. 在设备A上导出数据（JSON格式）
2. 将文件传输到设备B
3. 在设备B上导入数据

### Q6: 折线图为什么没有数据？

折线图记录的是历史市值变化：

- 每次添加或修改持仓时，系统会记录当天的总市值
- 需要使用一段时间后才会有历史数据
- 系统最多保留30天的历史记录
